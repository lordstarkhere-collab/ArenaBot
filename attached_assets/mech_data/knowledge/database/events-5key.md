|  | How Fortune Vaults work |  |  |  |  |  |  |
|  | The population of skins offered in fortune vaults is driven by the unlocked mechs you can obtain blueprints for (Mechs owned plus mechs in unlocked Gear Hub tiers) |  |  |  |  |  |  |
|  | Skins are grouped into categories for which all invididual skins have the same % chance to be obtained. |  |  |  |  |  |  |
|  | Individual skin % chance = Category total chance / number of skins in that category |  |  |  |  |  |  |
|  |  | Note: There is a category that conmingles both rare and epic skins. It is designated here as "basic rare/epic skins" category to be distingiushed from the rare skins category and the epic skins category. |  |  |  |  |  |
|  | Obtaining skins from one category only increases the chance of getting the remaining skins within that category - does not change the chance of getting resources or skins from other categories. Only after depleting all skins from a category the % chance of resources or other skin categories changes. |  |  |  |  |  |  |

|  | Q: Is it better to unlock the skins using the 5-key vault and then get better rewards in the 10-key vault? |  |  |  |  |  |  |
|  | A: Let's calculate to find out.   If you have X rare skins, and you only open the 10-key vault you'll need on average X/0.3 vaults to deplete that category, which will cost you X*33.33 keys. While opening these vaults you'll win on average X*33.33*3.72 = X*124 A-Coins  Alternatively, you can open on average X/0.299 5-key vaults to deplete that category. That's  X*16.72 keys that earn you 0 A-Coins. But then your next X*16.61 keys will yield 5.31 A-Coins per key or X*88.2 A-Coins  So when you compare both scenarios after X*33.33 keys you'll have depleted the rare skins. The only difference is that by opening the 5-key vaults instead of the 10-key vaults you're losing on average 35 A-Coins and 0.95 epic skins for each rare skin you have to unlock.  You can repeat this calculation for the basic rare/epic skin category and get the same result - it's better to open the 10-key vault.  TL;DR : the answer is no, it is not better to use the 5-key vault to unlock skins. If you really want to speed things up to get more A-Coins from the vaults you can buy the 25k-75k credits rare skins for for the mechs you own. |  |  |  |  |  |  |
|  | Q: When is it ever worthwhile to open the 20-key vault? |  |  |  |  |  |  |
|  | A: The 20-Key vault is better than the 10-key one in a few cases:  1. You've depleted all skins (including legendary ones) 2. Your enjoyment of the high risk/high reward gamble is worth more to you than actually getting more A-Coins long-term 2. You need to obtain legendary skins for an event whose reward is better than the loss of value for the keys you spend 4. You need to win 2,500 A-Coins for an event reward and the tiny chance of getting it from the vault is worth more than the loss of value for the keys you spend 5. You have depleted all rare and epic skins and you are willing to sacrifice some A-Coins in the short term to get better fortune rewards in the long-term.   How long would it take to break even for depleting legendary skins? If you have Z legendary skins to unlock, it would take you Z/0.495 vaults or Z*40.4 keys to unlock all legendary skins. You're losing 9.29-6=3.29 A-Coins for each of those keys, or Z*132.93 A-Coins. After depleting legendary skins you'll gain an additional 11.89-9.29=2.6 A-Coins for each key, so it would take you another Z*132.93/2.6 =Z*51.1 keys to break even. An overall investment of Z*91.5 keys to break even and start gaining more A-Coins.  Keep in mind that if during that time new skins/mechs are added to the game you'll need to deplete those as well which would take even longer. |  |  |  |  |  |  |
